# clean-disk

Lightweight disk cleanup script for Windows using Command Prompt (CMD).

## Overview

This project provides a simple and efficient way to remove temporary files and free up disk space on Windows systems. It uses native CMD commands to perform cleanup tasks without requiring external dependencies.

## Features

- Removes temporary files from user and system directories
- Clears Windows temporary folders
- Flushes DNS cache
- Deletes unnecessary log files
- Lightweight and easy to use

## Usage

1. Download or clone the repository
2. Run the script as administrator
3. Wait for the cleanup process to complete

## Requirements

- Windows operating system
- Command Prompt (CMD)
- Administrator privileges (recommended)
